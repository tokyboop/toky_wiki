# 2026-04-08 | 工具发现：微信公众号文章批量导出

## 背景

偶然发现 wechat-article-exporter，解决了"想批量保存公众号文章"但一直没有好工具的问题。

---

## 工具基本信息

- **仓库**：https://github.com/wechat-article/wechat-article-exporter
- **在线版**：https://down.mptext.top
- **文档**：https://docs.mptext.top
- **技术栈**：Nuxt + Vue 3 + TypeScript，可部署到 Cloudflare Workers / Docker
- **规模**：⭐ 8.4k，2024年7月上线，持续活跃

---

## 核心原理

微信公众号后台编辑文章时，有"搜索引用其他公众号文章"的功能，该接口对所有已认证公众号开放。exporter 复用这个接口实现对任意公众号文章的批量抓取。

**关键点**：使用的是微信官方接口，不是破解，相对稳定。

---

## 使用前提

需要一个**微信公众号账号**（个人订阅号即可，免费，实名注册后空号即可用）。网站不提供共享账号，每次用自己的账号扫码登录。

---

## 能做什么

| 功能 | 说明 |
|------|------|
| 搜索任意公众号 | 关键字搜索 |
| 多格式导出 | HTML / MD / DOCX / Excel / JSON / TXT |
| HTML 100% 还原 | 图片和样式全部打包，离线可读 |
| 文章过滤 | 按作者、标题、时间、合集筛选 |
| 合集批量下载 | 支持 |
| 评论/阅读量导出 | 需额外抓包获取 credentials |
| 开放 API | 自建后可程序化调用 |

---

## 使用方式

**最简路径**（推荐）：直接用在线版，无需任何部署。

1. 打开 https://down.mptext.top
2. 用公众号账号扫码登录
3. 搜索目标公众号 → 选文章 → 选格式 → 下载

**进阶**：Docker 自建 + 调 API，可实现定期抓取、自动存档等自动化需求。

---

## 局限

- 批量下载太快可能被微信限频
- Claude 无法替代扫码登录步骤，自动化的上限是"登录后的 API 调用"
- 评论/阅读量数据获取门槛较高（需抓包）

---

## 适用场景

- 批量归档关注的公众号文章
- 将文章导入 Notion / 本地知识库
- 研究某公众号的历史内容
