## 8 个隐藏功能速览

### 1. Buddy — AI 宠物
- 18 种物种，5 个属性，有稀有度（普通 60% → 传说 1%，1% 概率出闪光版）
- 由用户 ID 哈希决定，无法刷稀有度
- 据说 5 月上线

### 2. Kairos — 主动模式
- 全新交互模式，Claude 变成 24h 在线主动助手
- 有自己的每日日志
- 专属工具：给你发文件、推送通知等

### 3. Auto Dream — 后台做梦
- 已灰度（`/memory` 可开启）
- 不用时在后台自动整理记忆，避免记忆紊乱
- → 详见 `knowledge/claude-code-auto-dream.md`

### 4. Daemon — 后台守护进程
- Claude Code 变成像数据库/Web 服务器一样的后台服务，常驻运行
- 属于 OpenClaw（小龙虾）特性

### 5. UDS Inbox — 跨会话通信
- 不同 Claude Code 窗口之间可以互相发消息
- 以前多窗口完全隔离，现在可以互相感知通信

### 6. Teleport — 跨机器传送
- 把整个工作会话（含完整上下文）打包，从一台电脑传送到另一台
- 场景：公司电脑干到一半，回家继续，不用背电脑

### 7. Ultraplan — 30 分钟远程规划
- `/ultraplan + 描述`，云端启动远程实例，用 Opus 4.6
- 最多 30 分钟：探索项目、分析可行性、规划步骤、提前避坑

### 8. Ultrareview — Bug 舰队审查
- `/ultrareview`，云端启动 Bug 舰队
- 默认 5 个 Agent，最多 20 个，每个从不同角度审查代码
- 耗时 10～25 分钟，暴力美学
